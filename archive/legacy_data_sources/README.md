# Legacy Data Sources Archive

This folder preserves the pre-refactor multi-source ETHPredict assets.

Active code now uses public Lighter OHLCV exports only. The archived files here
are kept for reference or future reintroduction of non-Lighter sources, but
active loaders should not read from this directory.

## Archived Sources

- Binance-style historical ETHUSDT OHLCV samples.
- DeFiLlama ETH TVL and protocol snapshot CSVs.
- Santiment ETH social/network metric CSVs.
- Previous multi-source feature and pipeline modules.
- Previous multi-source README and Lighter integration notes.
- Previous experiment-runner config/schema sections for DEX simulation, bribe optimization, and parameter optimization.
